# 🌊 R.I.A.M - Robô Inteligente Anti Derrame Marítimo

## Sobre o Projeto
Sistema automatizado para deteção, alerta e mitigação de derrames de petróleo em alto mar utilizando sensor de fluorescência, GPS, LoRa e um sistema de biorremediação para degradação de hidrocarbonetos.

## Funcionalidades
- 🔬 Deteção de petróleo por sensor de fluorescência (4 bandas espectrais)
- 📡 Comunicação LoRa (longa distância, baixo consumo)
- 📍 Posicionamento GPS em tempo real
- 🧫 Sistema de biorremediação com dispersão automática
- 📊 Dashboard web com monitoramento em tempo real
- 📋 Histórico de derrames e estatísticas
- 🗺️ Identificação de áreas marítimas

## Tecnologias
- **Backend:** Python, Flask, Flask-SocketIO
- **Frontend:** HTML5, CSS3, JavaScript, Chart.js
- **Banco de Dados:** MySQL
- **Hardware:** ESP32, LoRa, GPS, Sensor de Fluorescência
